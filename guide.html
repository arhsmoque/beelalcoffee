🔐 Logging In

Open admin.html on your phone or computer. You will see a PIN entry screen.

🏪
Owner PIN
4-digit PIN set by your developer

Access to: Store settings, Menu management, Orders, AI Studio.

🛠️
Developer PIN
Set by the developer on first run

Everything above, plus: Dev tools, AI model settings, error logs, master theme snapshot.

Tip: If you enter the wrong PIN, wait 2 seconds and try again. PINs are stored securely in the database — not in the browser.

🏪 Opening & Closing the Store

The Store tab controls whether customers see an "Open" or "Closed" banner on the menu page.

Toggle Open / Closed

1
Log in with your Owner PIN.
2
Tap the Store tab (first tab at the top).
3
Press Open Store or Close Store — the change takes effect immediately for all customers.
Important: This does not stop customers from placing orders — it just shows a warning banner. You can still receive WhatsApp orders when "closed".

Updating Store Info

In the Store tab you can edit:

Store NameDisplayed at the top of the menu
SloganShort tagline below the name
Opening HoursShown in the menu header bar
WhatsApp NumberCountry code + digits only, e.g. 60122203743
Logo URLOptional — link to an image (HTTPS)

After making changes, tap Save Store Info. Changes are visible to customers immediately.

📋 Managing the Menu

The Menu tab lets you add, edit, delete, and mark items as sold out.

Mark an Item as Sold Out

1
Find the item in the menu list.
2
Tap Sold Out. The item stays visible on the menu but is greyed out with a "Sold Out" badge — customers cannot order it.
3
Tap Enable to make it available again.

Add a New Item

1
Tap + Add Item at the top of the Menu tab.
2
Fill in the item name, category, emoji, and description.
3
For drinks: enter prices for Hot, Cold, and/or Frappé sizes. Leave a size blank to hide it.
For food: enter a single fixed price.
4
Optionally add a photo (JPG or PNG — it will be resized to 500px automatically).
5
Tap Add Item to save.

Edit an Item

Tap Edit next to any item. Change the fields you want, then tap Save.

Delete an Item

Tap Del next to an item, then confirm. This cannot be undone.

Manage Categories

Categories appear as horizontal tabs on the menu page. In the Menu tab, scroll down to the Categories section to:

  • Edit a category's name or emoji
  • Delete an empty category (categories with items cannot be deleted)
  • Add a new category
Tip: Category order on the menu follows the order they appear in the Admin categories list.

Add-ons & Options

When editing an item, scroll to Options / Add-ons. Two types are available:

Choice (required)

A dropdown the customer must pick before adding to cart — e.g. sauce type, spice level. No extra price.

Optional (checkbox + price)

A checkbox the customer can optionally tick — e.g. "Extra Cheese". Set the extra price in the price field.

📦 Viewing Orders

The Orders tab shows a log of all orders customers sent via WhatsApp, newest first.

Each order entry shows: customer name, timestamp, items ordered (with sizes), and total amount.

Tip: This is a read-only log for reference. Actual order fulfilment happens through WhatsApp — the customer sends the order message and you reply or prepare from there.

Tap Refresh to reload the latest orders. Orders are stored for up to 1,000 entries before the oldest are removed.

🤖 AI Studio

AI Studio (available with the Developer PIN) lets you change the look of the menu — colours, fonts, theme — using plain-language instructions or by uploading a design template.

Requires: An OpenRouter API key. Set it once in the Dev tab → API Key field. Free-tier keys are available at openrouter.ai.

Changing the Theme with AI

1
Tap the AI Studio tab.
2
Type a request in plain language — for example: "Make it look festive for Christmas — red and gold" or "Change the background to dark navy".
3
Tap Send and wait a few seconds. The AI will apply the changes and explain what it did.
4
If you like the result, tap Save Theme. If not, tap Restore Last Saved to go back.

Quick Action Chips

Use the shortcut chips below the chat input for common actions:

🌸 Seasonal Theme 🎉 Festive Theme ↩ Reset to Default ✏️ Change Slogan 🚫 Category Off ✅ All Available

Upload a Design Template (ZIP)

If you have an HTML template ZIP file (e.g. from a design marketplace), you can upload it to extract its fonts and colour palette, then apply them to your menu.

1
Scroll to Template Upload in AI Studio.
2
Tap Upload ZIP and select your file.
3
The extracted fonts and colours are shown. Choose which elements to update, then tap Apply Selected.

Factory Reset

To restore everything to the original theme set up by your developer, tap Factory Reset in AI Studio. This reverts to the master snapshot taken on first setup.

🔒 Security & PINs

Change Owner PIN

1
Go to the Security tab.
2
Enter the new 4-digit PIN in both fields, then tap Save Owner PIN.
Write it down. If you forget both the Owner PIN and Developer PIN, contact your developer — they can reset it via the Firebase database directly.

Change Developer PIN

Only the developer can change the Developer PIN. It is done from the Security tab when logged in with the Developer PIN.

🛟 Recovery & Troubleshooting

🔑
Forgotten Owner PIN

Log in with the Developer PIN → Security tab → reset the Owner PIN.

🎨
Theme Looks Wrong / Broken

Log in with Developer PIN → AI Studio → tap Factory Reset. This restores the original theme.

🌐
Page Not Loading / Network Error

Check your internet connection. The menu and admin panel require a connection to Firebase. If the problem persists, contact your developer.

🐛
Viewing Error Logs

Log in with Developer PIN → Dev tab → scroll to Error Log. The last 20 errors are shown with timestamps and details.

🔄
Redeploying / Updating the App

Push updated files to the GitHub repository. Cloudflare deploys automatically in ~30 seconds. Database data is unaffected by redeployments.

Need help? Contact your developer.
